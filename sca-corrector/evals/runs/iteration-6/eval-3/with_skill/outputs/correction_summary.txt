==================================

2. Estudiar algoritmos de optimización:

   - Enfoque greedy: Selecciona en cada paso el usuario que cubre más módulos sin cubrir

   - Considerar agregar logging o output de debug

5. Estructura propuesta para Parte B (pseudocódigo):
   ```
   uncovered_modules = {todos los 8 módulos}
   selected_users = []
   
   while uncovered_modules is not empty:
       best_user = null
       best_count = 0
       
       for each user:
           count = modules_covered_by_user AND uncovered_modules
           if count > best_count:
               best_user = user
               best_count = count
       
       selected_users.add(best_user)
       uncovered_modules.remove(modules_covered_by_best_user)
   
   return selected_users
   ```
